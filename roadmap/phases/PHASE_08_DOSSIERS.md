# Phase 08 — Dossiers

## Purpose

Create clean numeric-first symbol dossiers.

## Keep from ASC

- strong numeric score stacks
- identity/classification
- market state
- broker specs
- rank context
- publication health
- missing data visibility

## Drop/compress from ASC

- repeated diagnostic blocks
- long taxonomy paragraphs
- irrelevant unavailable spam
- repeated trade permission lines in every mini-block
- route diagnostics unless contradiction exists

## Required sections

- header
- account exposure
- market state
- broker specs
- surface ranking
- bucket ranking
- deep review
- missing/degraded
- operator summary
