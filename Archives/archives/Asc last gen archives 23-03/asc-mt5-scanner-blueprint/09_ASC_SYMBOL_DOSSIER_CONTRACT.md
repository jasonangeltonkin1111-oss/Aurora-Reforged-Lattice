# ASC Symbol Dossier Contract

## Purpose

The dossier is a composed operator surface.
It must not become the birthplace of scanner truth.

## Dossier composition order
- Layer 1 market-state block
- Layer 2 packet block
- Layer 3 eligibility block
- Layer 4 shortlist / promoted-set block
- Layer 5 deep packet block

## Frequent dossier updates

Constant rolling dossier hydration belongs only to the most relevant promoted symbols.
Focused-symbol HUD motion alone is not enough reason for same-speed dossier writes.

## Wording law

Use operator language.
Do not leak internal scheduler jargon or pretend a later layer is active when it is not.
